Current approaches

  • Text approach
    • Changing meanings of words
    • Some words dominating the field, e.g. design, user, etc.
  • Network approach
    • Focus usually on collaboration networks
  • Availability of data
    • Other databases not very helpful
    • Or require quite some manual scraping

Scope

  • Gather the references of all CHI papers from 1981 to 2019
    • Digital Library ID useful for identifying papers
  • Construct the citation network within >6000 papers
    • Well defined boundary
  • Cluster the papers
    • Using methods for social networks
  • Examine the clusters
    • Do they represent different topics/themes?
    • How is the trend over the years?

Community detection

  • Algorithms for clustering relational data
  • Principles based on heuristics
  • Pros
    1. Quick
  • Cons
    1. Huge inequality in cluster sizes
    2. Variation of results between algorithms
    3. Number of clusters?
    4. Resolution limit

Stochastic block models

  • Statistical models for clustering relational data
  • Properly defined by model assumptions
  • Pros
    1. One definitive set of results (theoretically)
    2. Can resolve issues with community detection
  • Cons
    1. Not off the shelf
    2. Slow / difficult to scale

Difference

  • Community detection
    • Nodes (papers) are tightly connected within each cluster
    • Low connectivity between clusters
  • Stochastic block models
    • Same connectivity for any pair of nodes in two clusters
    • Not guaranteed to be high or low
    • Existence of miscellaneous group

Statistical models

  • Slightly different way of looking at the data
    • Computational/physical: input the parameters, output simulated data
    • Statistical: input real data, output (estimates of) the parameters
  • Linear regression
    • The slope and intercept
  • Epidemic modelling
    • The transmission rate
    • The basic reproduction number
    • Say something about when individuals get infected

Inferring hidden structure

  • is not trivial
  • Always comes with some degree of uncertainty
  • Label switching
    • (A, B, C) in group 1; (D, E) in group 2
    • (D, E) in group 1; (A, B, C) in group 1

Utilising extra information

  • Publication years available
  • Forcing the mean year to be increasing for groups
    • Earlier groups represent earlier topics
    • The group labels are actually representative

A new direction

  • Directed Acyclic Graph (DAG)
    • No cycles of any length
    • Start with any node A and travel along the edges
    • Impossible to go back to node A

(Source: Wikipedia)
