Preferential attachment model (Barabási and Albert, 1999)

  • Each new node enters the network one by one, with \(\mu\) new edges
  • Each existing node gets an edge with probability proportional to current degree

Ingredients

The epidemic

  • Susceptible-Infected (SI) model
  • Markovian, constant infection rate \(\beta\)

The underlying network \(G\)

  • Some sort of preferential attachment model: \((\mu,~\gamma)\)
  • \(G_{ij}=1\) if \(i\leftrightarrow j\), \(0\) otherwise

The transmission tree \(P\)

  • \(P_{ij}=1\) if node \(j\) is infected by \(i\), \(0\) otherwise
